Dart 库

一. 全平台

  1. dart:async 
支持使用Future和Stream类实现异步编程


    2. ```
       dart:collection
       提供dart:core中不支持的额外的集合操作工具类
    

  2. dart:convert
转换不同数据的编码器和解码器,包括JSON和UTF-8


    4. ```
       dart:core
       内置类型、集合以及其它的一些核心功能
    

  3. dart:developer
开发工具,只支持JIT和dartdevc


    6. ```
       dart:math
       算术相关函数及常量,随机数生成器
    

  4. dart:typed_data
高效处理固定大小数据和SIMD数字类型的列表


    #### 二. 原生平台

    1. ```
       dart:ffi
       使用原生C语言API
    

  5. dart:io
支持非Web应用的文件,Socket HTTP 及 其它I/O操作


    3. ```
       dart:isolate
       使用Isolate实现并发编程:类似于线程独立的Worker
    

  6. dart:mirrors
支持检查和动态调用的基本反射功能;仅在JIT中有效,Flutter无效


    #### 三. Web平台

    1. ```
       dart:html
       HTML元素和其它资源
    

  7. dart:indexed_db
客户端使用的可以索引的键对值存储


    3. ```
       dart:js 弃用
    

  8. dart:js_util
补充功能性API


    5. ```
       dart:svg
       用于可缩放的矢量图形

  9. ```
    dart:web_audio
    浏览器的高保真音频编程


    7. ```
       dart:web_gl
       浏览器3D编程
